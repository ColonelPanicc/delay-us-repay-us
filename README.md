# DelayUsRepayUs

Delay us repay us - bad idea. Fun execution

## Developer Onboarding

Install `prek` ([docs](https://prek.j178.dev/)).

```bash
cargo install --locked prek
```

Install a git commit hook to run `prek` hooks

```bash
prek install
```

Try running `prek` hooks manually

```bash
prek run
```
